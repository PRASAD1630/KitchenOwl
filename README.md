# 🍽️ KitchenOwl - Full Stack Smart Kitchen Management System

A modern full-stack kitchen management application designed to simplify meal planning, grocery organization, and recipe management. The platform enables users to efficiently manage their pantry, shopping lists, and recipes through an intuitive and responsive interface.

> **Note:** This repository is maintained for learning and full-stack development practice.

---

## 🚀 Features

- 🛒 Smart Grocery List Management
- 📖 Recipe Management
- 🍳 Meal Planning
- 👨‍👩‍👧 Multi-user Support
- 📱 Cross-platform Application
- ⚡ Responsive User Interface
- 🔄 Real-time Data Synchronization

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/PRASAD1630/KitchenOwl-full-stack.git
```

### Navigate to the project

```bash
cd KitchenOwl-full-stack
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
